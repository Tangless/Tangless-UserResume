##运行方法
1.直接用浏览器打开src目录下的index.html
2.用node.js搭建一个本地的服务器，命令行执行 node text.js 然后在浏览器中输入：http://localhost:8888/src/views/index.html
注意test.js文件中的documentRoot文件路径，运行时要改为文件对应的路径.
